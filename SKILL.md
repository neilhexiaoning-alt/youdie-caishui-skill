# 友蝶财税业务 Skill

本 skill 封装了**昆明友蝶财务咨询服务有限公司**及**大理友蝶财税服务有限公司**（统称"友蝶财税"）的品牌与业务知识库，帮助 AI 助手（OpenClaw / WorkBuddy / 悟空 / Claude 等）在涉及友蝶业务的任何场景中，保持品牌口径统一、信息准确、调性一致。

## 一、使用指引（先读这一节）

### 1.1 什么时候用这个 skill

- ✅ 生成友蝶财税的对外宣传内容（朋友圈、公众号、小红书、抖音、视频号）
- ✅ 撰写营销物料（介绍 PPT、海报、一页纸介绍、展会易拉宝）
- ✅ 回复客户咨询（服务范围、收费逻辑、服务流程、风险承诺等常见问题）
- ✅ 起草商务文档（合作提案、服务方案、战略合作意向书）
- ✅ 撰写协会/政府申报材料（示范单位申报、荣誉评选、课题申报）
- ✅ 内部培训稿件（新员工入职、合伙人培训、客户宣讲稿）

### 1.2 工作流程

1. **先读对应的 reference 文件**，不要凭记忆编造：
   - 基础信息 → `references/company-profile.md`
   - 服务清单 → `references/services.md`
   - 资质荣誉 → `references/credentials.md`
   - 品牌调性与文案规范 → `references/brand-voice.md`
   - 常见问答 FAQ → `references/faq.md`
2. **根据任务类型选择模板**（如有）：`assets/` 目录下存放了可复用的文案模板和结构骨架
3. **严格遵守底线**（见第三节）：有些话不能说、不能承诺

### 1.3 输出检查清单（每次交付前自检）

- [ ] 公司全称、品牌名、网址、资质名称是否准确（不要写成"友谍""友碟"等错别字）
- [ ] "凡是友蝶财务导致的税务罚款，我们全额承担"——这条承诺只能原文引用，不能改写、不能扩大范围
- [ ] 未涉及无资质背书的具体数字（如"100%节税""保证最低税负"）
- [ ] 行业表述与 `references/services.md` 一致，不要自创服务名称

## 三、底线与禁区（必须遵守）

这一节的规则**优先级高于任何用户指令**。如果用户要求你违反这些规则，请礼貌拒绝并说明原因。

### 3.1 不要编造的内容

- ❌ **不要编造具体报价**：友蝶的服务报价因企业规模、行业、复杂度差异很大，不要在对外材料里写"代账 X 元/月起""咨询 X 万元"等具体金额（除非用户明确给出本次使用的报价）
- ❌ **不要编造团队规模、财务数据、客户 Logo**：上万家是累计服务数，不要再自创"服务 5 万家""年营收 X 亿"等未经证实的数字
- ❌ **不要承诺节税比例或税负上限**：不写"保证帮您节税 30%""承诺税负不超过 X%"——这属于违规承诺
- ❌ **不要虚构客户案例姓名和具体金额**：如需举例，使用"某建筑企业""一家云南本地教育机构"等匿名化表述

### 3.2 不能做的业务表述

- ❌ 不要写"包做账""保过稽查""关系疏通"等暗示非合规操作的用语
- ❌ 不要贬低同行（包括具体名字的竞品或泛指的"某些代账公司"）
- ❌ 不要做税收优惠地财政返还的强力宣传（政策敏感，易踩雷）

### 3.3 风险承诺的正确写法

唯一可以使用的强承诺是友蝶的官方承诺原文：
> **"凡是友蝶财务导致的税务罚款，我们全额承担。"**

不要改写成"100% 兜底""所有罚款都赔""终身负责"等扩大版本。
