开始安装
只需要 3 步
1

下载 skill 到本地

打开终端（Terminal），粘贴下面这条命令，回车即可。
文件会下载到当前目录下的 youdie-caishui-skill 文件夹。
git clone https://github.com/neilhexiaoning-alt/youdie-caishui-skill.git
💡 没装 git？
Mac 用户打开终端输入 git --version，系统会提示安装；
或直接访问 GitHub 页面点击 Code → Download ZIP 下载压缩包。
2

选择你的 AI 平台

以下 4 个平台都已适配，找你在用的那个按说明接入即可：
🐉
OpenClaw
开源 Agent 平台
  • 整个目录移动到 ~/.openclaw/skills/
  • 重启 OpenClaw
  • 对话提到"友蝶"自动触发
🤝
WorkBuddy
企业级工作助手
  • 新建 Knowledge Space "友蝶财税"
  • 上传 SKILL.md 为入口
  • 上传 references 和 assets
🐵
悟空
国产 AI Agent 平台
  • SKILL.md 作为系统提示词
  • references 导入为关联文档
  • 开启"按需加载"能力
🤖
Claude Code
Anthropic 官方 CLI
  • 整个目录放到 ~/.claude/skills/
  • 重启 Claude Code
  • 对话中自动识别触发

常见问题
遇到这些先别慌
Q1下载失败 / 报错 "could not resolve host"？
A网络问题。可以换个网络环境，或直接去 GitHub 页面点 Download ZIP。
Q2AI 助手说没找到这个 skill？
A检查三件事：目录路径是否正确、是否重启了 AI 平台、对话中是否明确提到 "友蝶" 关键词。
Q3我想修改 skill 内容（比如加新的服务）？
A直接编辑 references/ 或 assets/ 下对应的 Markdown 文件即可，改完保存下次对话就生效。
Q4AI 生成的内容里出现了不准确的信息？
A先核对 references/ 里的源头数据是否正确；如果源头准确但 AI 写错，在对话中明确指出并要求修正。
Q5这个 skill 是开源的吗？能商用吗？
A代码和文档结构 MIT 协议，可自由复用；但"友蝶"品牌信息、商标为公司专有，他人不得直接套用推广自己业务。
