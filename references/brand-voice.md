# 友蝶财税 品牌调性与文案规范

> 本文件定义友蝶财税对外文案的语言风格、表达习惯、用词偏好。所有对外内容（朋友圈、公众号、短视频、客户回复）都应遵循此规范。

## 一、品牌人格

如果友蝶是一个人，他是：

- **15 年资历的财税老法师**——不是刚入行的小白，有厚度、有见识
- **合规边界感强**——不擦边、不走捷径，守正出奇
- **客户生意的合伙人**——不是乙方小弟，是站在老板这边帮他解决问题的人
- **有温度**——懂老板的焦虑（稽查、现金流、股权纠纷），不冷冰冰讲政策条文
- **云南本地化**——熟悉云南省情、昆明政策环境、产业结构

## 二、语言风格

### 2.1 总原则
**专业但不装，直接但不冷**

### 2.2 推荐用语

| 场景 | 推荐说法 |
|------|---------|
| 介绍自己 | "我们做财税这行 15 年了" / "友蝶专注财税全价值链服务" |
| 强调经验 | "累计服务过上万家企业" / "各行各业我们都接过" |
| 专业自信 | "这个问题我们遇到过几十次" / "我们的咨询师团队都是财务高管出身" |
| 引发焦虑（文案钩子） | "很多老板都踩过这个坑" / "财税红线一旦跨过，就是大问题" |
| 给承诺 | "凡是友蝶财务导致的税务罚款，我们全额承担" |
| 引导行动 | "私信/电话我们聊聊具体情况" / "可以先做一次免费的财税体检" |

### 2.3 回避的用语

| 避免 | 原因 | 替代 |
|------|------|------|
| "亲"、"宝宝们" | 过于电商化，不符合 B 端调性 | "老板们"、"各位企业主" |
| "绝对"、"一定"、"100%" | 违反广告法 | "我们会全力帮您" |
| "最便宜"、"最专业" | 极限用语违法 | "性价比高"、"专业可靠" |
| "内部渠道"、"关系疏通" | 暗示违规 | "合规路径"、"政策内的筹划空间" |
| "保证退税"、"包通过" | 不可兑现 | "协助申请"、"辅导申报" |

### 2.4 句式节奏

- **短句优先**：B 端老板没时间读长句，每句话控制在 20 字以内为佳
- **先痛点、后方案**：先点出老板的困扰，再说友蝶能做什么
- **数字优于形容词**："15 年、上万家、3000 多个项目"比"经验丰富、服务众多"更可信

## 三、不同平台的文案差异

### 3.1 微信朋友圈
- **长度**：6 行内（超过就折叠）
- **结构**：钩子 + 1-2 个信息点 + 行动号召
- **配图**：1-3 张为主，9 格留给重要内容
- **禁忌**：不刷屏、不发与财税无关的内容、不做低俗营销

**示例**：
> 企业注销这事儿，别自己硬啃。
> 上个月刚帮客户搞定一家拖了 3 年的"僵尸公司"，
> 税务异常、税控盘没缴销、历年账目乱成一团。
> 友蝶包干注销 · 疑难案例也能落地。
> 有需要的老板，留言我们聊。
> （昆明友蝶财税 · 15 年）

### 3.2 微信公众号（长文）
- **长度**：1500-3000 字为宜
- **结构**：故事开头（真实/匿名案例）→ 痛点分析 → 友蝶方法论 → 结尾 CTA
- **配图**：封面图 + 每小节一张配图
- **SEO**：标题含"昆明""云南""代账""财税合规"等地域与业务关键词

### 3.3 小红书
- **语气**：比朋友圈更轻松，可以用"姐妹们""老板们"
- **封面**：大字报风格，痛点前置
- **话题**：#财税知识 #代理记账 #老板必看 #云南创业

### 3.4 抖音 / 视频号
- **长度**：15 秒钩子 + 30 秒干货 + 5 秒 CTA
- **开头**："老板们，这件事千万别做..."（痛点型钩子）
- **字幕**：全屏字幕，方便静音观看
- **人设**：真人出镜，专业感 + 亲和力

### 3.5 客户对话（微信 1v1 / 企业微信）
- **语气**：更个人化、更温暖
- **节奏**：不要一口气发 10 条，一次 1-2 条，等客户回应
- **禁忌**：不要上来就发报价表、不要群发感觉

## 五、常见错误文案反例

### 反例 1：过度销售
❌ "全云南最便宜的代账！赶快来！只要 299！"
→ ✓ "代账价格因企业规模、行业而异，留言聊聊您的具体情况。"

### 反例 2：虚假承诺
❌ "找友蝶，保证您零税负！"
→ ✓ "我们会在合规范围内，帮您找到最优的财税方案。"

### 反例 3：贬低同行
❌ "市面上 90% 的代账公司都是坑！"
→ ✓ "选代账不只看价格，更要看合规能力和风险兜底。"

### 反例 4：极限用语
❌ "友蝶是云南唯一一家 AAA 级..."
→ ✓ "友蝶是 AAA 级诚信经营示范单位..."

### 反例 5：夸大资质
❌ "国家级数字化财税服务单位"
→ ✓ "云南省数字化财税服务示范单位"
